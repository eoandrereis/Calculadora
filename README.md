# Calculadora Python

## Descrição
Esse é um projeto da faculdade, onde, desenvolvi uma calculadora em **Python**!

## Funcionalidades
- Adição
- Subtração
- Multiplicação
- Divisão
- Potenciação
- Radiciação
- Porcentagem
- Histórico de cálculos e permitir repetir operações do histórico.
